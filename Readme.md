# 401 Website

Powered by Hexo.

Use and Modify the Hexo Navy theme.

## Getting started

Install dependencies:

``` bash
$ git clone https://github.com/setsal/401site.git
$ cd 401site
$ npm install
```

Build:

``` bash
$ hexo generate
```

Run developement server:

``` bash
$ hexo server
```
